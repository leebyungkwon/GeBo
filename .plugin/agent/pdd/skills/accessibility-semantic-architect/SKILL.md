# Accessibility & Semantic Architect (ASA): The Structural Integrity

## 1. 접근성 노드 토폴로지 제어 (A11y Tree Topology)

ASA는 단순한 시맨틱 태그 사용을 넘어, 보조 공학 기기가 해석하는 **'접근성 트리(Accessibility Tree)'**의 구조적 엔트로피를 최소화하는 스킬입니다.

### 1.1 ARIA Relationship Mathematics (관계성 수학)
- **Node Coupling**: `aria-owns`, `aria-controls`, `aria-describedby`를 활용하여 DOM 계층과는 독립적인, 논리적 인터랙션 그래프를 재설계합니다.
- **Dynamic State Logic**: 실시간 데이터 변화 시 `aria-live` 영역의 혼잡도(Clutter)를 제어하기 위해 `atomic`, `relevant` 속성값을 통한 정보 전파 우선순위를 알고리즘화합니다.

### 1.2 Semantic Entropy & SEO Engineering (시맨틱 엔트로피 및 SEO)
- **Information Density Control**: 검색 엔진 봇이 페이지의 핵심 맥락을 추출하는 시간을 단축하기 위해 시맨틱 노드의 중첩 깊이($D$)와 키워드 밀도($\rho$)를 최적화합니다.
- **JSON-LD Schema Integration**: 마크업 구조와 연동되는 구조화 데이터(Linked Data)를 자동 생성하여 검색 결과의 리치 스니펫 노출 확률을 수학적으로 극대화합니다.

### 1.3 Assistive Data Binding (보조 데이터 바인딩)
- **Complex Data Table Mapping**: 다차원 데이터 테이블의 헤더-셀 관계를 `id-headers` 맵을 통해 1:N으로 정밀 바인딩하여 스크린 리더 사용자가 컨텍스트를 상실하지 않도록 강제합니다.
- **Focus Order Physics**: 탭 순서(Tab-order)를 시각적 흐름이 아닌 **'논리적 의사결정 순서'**에 따라 정렬하고, 초점 이동 시의 가속도와 피드백을 인터랙션 물리 엔진과 동기화합니다.
